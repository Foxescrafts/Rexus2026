npm i
жди пока установятся библиотеки

после , открывай index.js
на 27 и 30 строке меняй токены на свои

и меняешь на айди группы, где должен работать бот

далее, открывай файл database.js + adatabase.js
и меняешь данные бд на свои

после снова открываешь консоль, и пишешь node index.js

ладно, если нормально;
1. устанавливаем на vds node.js
2. устанавливаем pm2 (npm install -g pm2)
3. переходишь в папку с проектом (ты же его загрузила по ftp?)
4. в папке с проектом (НА ВДС!!!) пишешь: pm2 start index.js
5. После пиши;
pm2 startup systemd
pm2 save
